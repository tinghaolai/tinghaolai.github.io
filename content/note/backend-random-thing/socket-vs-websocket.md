## Socket vs Websocket

### Socket

Socket is not a protocol, 
it's a interface,
when two service communicate with each other,
socket is the interface to comprise data to match protocol.

### Websocket

WebSocket is a protocol base on TCP protocol,
transfer data TCP protocol.
